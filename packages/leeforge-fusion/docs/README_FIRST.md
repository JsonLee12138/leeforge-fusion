## 🔍 快速搜索

使用浏览器搜索功能 (Ctrl+F 或 Cmd+F) 查找关键词：

- **路由**: 搜索 "路由" 或 "router"
- **守卫**: 搜索 "守卫" 或 "guard"
- **中间件**: 搜索 "中间件" 或 "middleware"
- **CLI**: 搜索 "CLI" 或 "命令"
- **SSR**: 搜索 "SSR" 或 "服务端渲染"
- **API**: 搜索 "API" 或 "接口"
- **部署**: 搜索 "部署" 或 "deploy"

## 🚀 立即开始

```bash
# 1. 安装
npm install @leeforge/fusion

# 2. 创建配置
# leeforge.config.ts

# 3. 创建页面
# app/page.tsx

# 4. 启动开发
npm run dev

# 5. 查看文档
# docs/GUIDE.md
```
